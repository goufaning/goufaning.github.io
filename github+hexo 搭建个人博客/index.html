github+hexo 搭建个人博客

发表于2016-07-23|更新于2021-06-22|Hexo
|字数总计:3.1k|阅读时长:10分钟|阅读量:

写在最前

一直想搭建一个个人博客，偶然发现github和Hexo可以比较简单的搭建一个免费的个人博客，而且网页非常漂亮。于是，上网参考了好多资料，终于搭建好了自己的个人博客。所以写这篇文章总结一下自己搭建博客的过程，也为想搭建个人博客的童鞋提供一点参考。

为什么搭建博客？

这个问题我相信每个人都有自己的想法，不过我还是推荐大家看看这篇文章：

  • 为什么你要写博客？

为什么选择GithubPages和Hexo

现在有那么多像CSDN、cnblog、新浪博客甚至QQ空间之类现成博客网站注册就可以使用，为什么我非要这么折腾呢？对于我来说，这样做最大的优点是这个博客是完全自由的，自己可以设计任意的主题，可以修改任何一个地方，并且可以绑定独立域名。另外，我非常喜欢markdown的写作方式，觉得它非常方便，非常便捷，非常优雅。当然，这些都只是工具，挑选自己喜欢的就可以了，博客中的文章才是最重要的。

域名配置

域名是网站的入口，也是网站的第一印象，比如大家熟知的百度域名是：百度 。

  1. 域名购买

    自己的个人博客当然希望拥有一个独特的域名啦，所以我们需要购买域名，这也是唯一我们需要花钱的地方。申请域名的网站有很多，这里我推荐用腾讯云 。域名常见的有com,cn,net,org等后缀，也有小众的xyz,me,io等后缀，当然常见的域名比如.com的会贵一点，小众的便宜一点。大家根据自己的喜好选就好。

    购买成功后按照网站提示逐步完成域名实名认证和域名信息，等待下发域名证书。（审核速度很快，实名信息上传到电子证书下发还不到半小时）关于备案：因为Hexo博客是托管在github上的，所以，除非你购买的域名是.cn后缀的，或者你给博客使用的某些插件需要部署在另外的云服务上这两种情况，否则是不需要备案的。

主题配置

如果你不喜欢Hexo默认的主题，可以更换不同的主题，在Themes 上有许多主题可以选择。我自己使用的是Butterfly 主题，你可以直接访问Butterfly主题官方文档进行主题配置,每个主题一般都有说明文档，大家可以按照文档个性化设置自己的博客。
